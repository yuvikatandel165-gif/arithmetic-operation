# Simple Calculator (Switch Case)

This is a simple arithmetic calculator built using:

- HTML
- CSS
- JavaScript (Switch Case)

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Modulus

## How to Run

1. Download all files:
   - index.html
   - script.js
   - style.css

2. Keep them in the same folder.

3. Open `index.html` in your browser.

## Description

The calculator takes two numbers and performs the selected arithmetic operation using a JavaScript switch case statement.
